# The Tordera Brine Plume — MITgcm simulation tour

Landing page for the QR code on the **ISMS / EOF 2026** (Cádiz, 8–10 July 2026) poster
*"Modelling the brine plume from the Tordera desalination plant outfall."*

It embeds a ~34 s guided tour of four MITgcm runs of the SWRO desalination brine plume
(NW Mediterranean): a flat channel and a realistic slope, each under modal (1 m/s) and
extreme / p95 (5 m/s) wind. ΔS = salinity excess (brine − no-brine run).

**Live page:** https://bcbayo-arch.github.io/tordera-brine-sim/
